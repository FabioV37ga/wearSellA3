CLIENTE:
1. criação de conta
	- email ou celular*
	- cpf*
	- nome*
	- senha*
1.1 - alterar / 1.2 - excluir / 1.3 - consultar

2. Comprar produto
	- Adicionar ao carrinho
	- Alterar carrinho
		excluir item;
		alterar qtd;
	- Vizualiar carrinho
	- Finalizar compra

3. Adicionar produto

SISTEMA:
1. adicionar produto ao banco de dados
2. verificar login
3. 

O sistema é um e-commerce, a plataforma é online, pode ser utilizada atraves de qualquer
navegador.

O sistema deve permitir o cadastro de clientes, incluindo as informações 
[cpf, nome, cel/email, senha]. Devem ser feitas verificações nos valores para garantir
que existam.

O sistema deve permitir o login de clientes, verificando o CPF(login) e senha no banco de 
dados.

A vitrine da loja deve ser disponível para todo e qualquer usuário, porém apenas usuários
logados podem adicionar itens ao carrinho.

Usuários logados poderão adicionar/remover/alterar itens no carrinho.

Em relação ao sistema, deve existir uma tela principal de itens, possuindo todos os itens.
Para cada item deve existir uma nova tela de compra, com seleção de tamanho e cor/variante.
